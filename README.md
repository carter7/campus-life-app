
# Campus Life App

A web application that centralizes campus events, maps, and an AI-powered student assistant.

## Quick Links (example)
- GitHub: https://github.com/carter7/campus-life-app
- Jira: https://carter7.atlassian.net/jira/software/projects/CLA/boards/1
- Confluence: https://carter7.atlassian.net/wiki/spaces/CLA
- Deployed: https://campuslifeapp-12345.uc.r.appspot.com

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB Atlas
- AI: OpenAI API (chatbot)
- Containerization: Docker
- Cloud: Google Cloud Platform (App Engine or Cloud Run)

## Local Setup (Docker)
1. Copy .env.example to .env and fill in variables (MONGO_URI, JWT_SECRET, OPENAI_API_KEY)
2. Run: `docker-compose up --build`
3. Backend: http://localhost:5000
4. Frontend: http://localhost:3000

## Docker & Deployment Notes
- Backend and frontend Dockerfiles provided.
- Use Google Cloud Build or Cloud Run to deploy containers.
